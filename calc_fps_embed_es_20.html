Calculadora ConsScale para Bots de Video Juegos de Acción en Primera Persona

Este artículo contiene una explicación detallada de cómo usar ConsScale para evaluar bots de video juegos FPS.
Vea más abajo como evaluar su agente (o vea aquí un ejemplo de evaluación).
También puede ver aquí una descripción de los niveles de ConsScale.

Calculadora ConsScale para Bots FPS: Formulario de adquisición de datos.

1.- Seleccione los componentes arquitectónicos.
2.- Indique el perfil cognitivo (formulario de habilidades cognitivas [CSi,j]).

Uso de la calculadora:
- Use las casillas de verificación que aparecen más abajo para seleccionar las características de su implementación.
- Los resultados aparecerán inmediatamente en el panel izquierdo.
- El gráfico del perfil cognitivo puede tardar hasta un segundo en aparecer.
- El botón [Nivel 3] carga el perfil de un agente adaptativo puro.
- Use el botón [Reset] para comenzar de cero (en vez de recargar página).
- Use el botón [Informe] para generar un informe completo de su evaluación.

Componentes Arquitectónicos del Agente:

E Entorno del juego de acción en primera persona.
B Cuerpo del jugador simulado por software.
Spropio El bot tiene un mecanismo de adquisición de datos de su estado interno (sensores propioceptivos).
Sext El bot tiene un mecanismo de adquisición de datos del entorno de juego (sensores exteroceptivos).
A El bot tiene mecanismos de control de su cuerpo (actuadores del jugador).
R El bot tiene un mecanismo para generar acciones basadas en la información sensorial adquirida (mecanismo de coordinación sensoriomotora).
M El bot tiene un mecanismo para almacenar (parte de) sus estados pasados (memoria).
   
Mn El bot es capaz de representar múltiples contextos en memoria.
Att El bot tiene un mecanismo para dirigir S y A hacia un Ei específico (mecanismo de atención).
SsA El bot tiene un mecanismo de evaluación de su propio estado interno.
I El bot tiene un mecanismo para representación de su propio yo.
O El bot tiene un mecanismo para la representación de otros yos.
AR El bot tiene un mecanismo para la comunicación precisa de sus estados mentales.
AVR El bot tiene un mecanismo de comunicación verbal precisa de los estados mentales.
Rn El bot tiene un mecanismo para ejecutar y sincronizar varios flujos concurrentes de conciencia.

Perfiles de Comportamiento (BPi,j) de los niveles ConsScale para bots FPS:


Nivel -1. Sin cuerpo definido (Disembodied) - L-1 = 0.00
N/A No se consideran BP a este nivel.

Nivel 0. Aislado (Isolated) - L0 = 0.00
N/A No se consideran BP a este nivel.

Nivel 1. Pre-funcional (Decontrolled) - L1 = 0.00
N/A No se consideran BP a este nivel.

Nivel 2. Reactivo (Reactive) - L2 = 0.00
BP2,1 Reflejos básicos: como la capacidad de retroceder cuando el bot choca o se bloquea con otro bot o con un obstáculo.

Nivel 3. Adaptativo (Adaptive) - L3 = 0.00
Seleccionar/Deseleccionar Todos
BP3,1 Habilidad para aprender nuevos comportamientos simples adaptados al juego: Comportamientos básicos que ayudan al bot alcanzar mejores puntuaciones, como disparar a otros jugadores cuando estos son detectados.
BP3,2 Habilidad para usar el propio estado (salud, munición, etc.) para aprender nuevos comportamientos adaptativos: Buscar botiquines (packs de salud) cuando el nivel de salud es bajo, o buscar munición cuando el bot se está quedando sin ella.

Nivel 4. Atencional (Attentional) - L4 = 0.00
Seleccionar/Deseleccionar Todos
BP4,1 Habilidad para ignorar la información sensorial que no es crítica para la tarea actual: Ignorar los kits de recarga de munición detectados cuando el bot está envuelto en un combate y no necesita más munición.
BP4,2 Habilidad para descartar las acciones inadecuadas para la situación actual: El bot evita acciones inútiles como disparar a las paredes cuando está huyendo de un enemigo.
BP4,3 Habilidad para seleccionar la información que merece la pena recordar (obtener de la memoria): Cuando el bot necesita munición, accede a su memoria para obtener la posición de los kits de recarga de munición vistos anteriormente. Se dirige directamente a recoger el más cercano.
BP4,4 Habilidad para evaluar a otros jugadores como amigos o enemigos. Habilidad para evaluar las ventajas de usar los diferentes packs de munición o salud: El bot no ataca a sus amigos. Las tareas de sanación (recuperación del nivel de salud) y rearmamento se realizan rápidamente seleccionando los mejores kits de munición y de salud.
BP4,5 Habilidad para seleccionar qué información debería almacenarse en la memoria: El bot almacena en su memoria la posición de los botiquines y de los kits de munición. El bot va directamente a las posiciones recordadas cuando necesita munición o recuperar su salud (ver BP4,3).
BP4,6 Habilidad para aprender mediante mecanismos de prueba y error: El bot identifica a otros jugadores como amigos o enemigos en base a prueba y error. Si un jugador considerado actualmente como amigo (ver BP4,4) comienza a atacar al bot, se pasará a considerarlo como enemigo y se desplegarán los comportamientos adaptativos correspondientes (huir o atacar).
BP4,7 Habilidad para adaptar el comportamiento dirigiéndolo a objetivos específicos: El bot muestra comportamientos dirigidos y sostenidos hacia sus enemigos, como perseguirlos, dispararlos continuadamente o huir de ellos.
BP4,8 Habilidad para evaluar el propio rendimiento en combate: Las acciones del bot que no contribuyen a las metas perseguidas son descartadas. Por ejemplo, el comportamiento de huida se cambia por otro cuando no contribuye a disminuir el daño recibido.
BP4,9 Habilidad básica para planificar los siguientes movimientos: El bot muestra una secuencia coherente de acciones planificadas para alcanzar una cierta meta. Por ejemplo, seguir buscando y recogiendo packs de salud hasta que el nivel de salud del bot es el máximo.
BP4,10 Habilidad para mantener una representación espacial relativa de los objetos del juego: El bot es capaz de localizar objetos de forma efectiva y calcular la posición relativa de los mismos independientemente de las posiciones cambiantes de su cuerpo y sus sensores (ver BP4,7). El bot muestra una precisión de tiro aceptable.

Nivel 5. Ejecutivo (Executive) - L5 = 0.00
Seleccionar/Deseleccionar Todos
BP5,1 Habilidad para intercalar la realización de diferentes tareas del juego: Algunos comportamientos del bot se interrumpen debido a ciertas circunstancias y luego se retoman (ver BP4,9)
BP5,2 Habilidad para perseguir la consecución simultánea de varias metas del juego: Por ejemplo, un combate se elude porque el bot necesita aumentar su nivel de salud, después de recuperar la salud, el bot continua el ataque.
BP5,3 Habilidad para evaluar el propio rendimiento en relación a la consecución de varias metas del juego: el bot estima hasta qué punto las metas se están alcanzando en base a las estrategias empleadas. Los comportamientos más efectivos se repiten con más frecuencia que los comportamientos que suelen conllevar resultados pobres.
BP5,4 Habilidad para aprender de las experiencia pasada en el juego: El bot utiliza la evaluación realizada según DCS5,3 para seleccionar las estrategias más prometedoras (ver BP5,1-3). Por ejemplo, el bot aprende a usar las armas más destructivas cuando las tiene a su disposición.
BP5,5 Habilidad para planificar acciones teniendo en cuenta todas las metas de juego activas: El bot intercala sus acciones de forma efectiva tal y como requiere el cumplimiento de las múltiples metas activas. Por ejemplo, el bot modifica ligeramente su trayectoria mientras está persiguiendo y disparando a un enemigo para recoger sobre la marcha los kits de munición disponibles en las inmediaciones.

Nivel 6. Emocional (Emotional) - L6 = 0.00
Seleccionar/Deseleccionar Todos
BP6,1 Habilidad para evaluar el propio estado como actor en el juego. Esta capacidad representa el aspecto funcional de las emociones: El bot entra en un estado determinado dependiendo de la evaluación que realiza del propio rendimiento.
BP6,2 Habilidad para adaptar los mecanismos de control al estado actual: El comportamiento global del bot se ve modulado por el estado emocional en el que se encuentra.
BP6,3 Habilidad para mantener una representación de las emociones tal y como se describen en BP6,1: Por ejemplo, si el nivel de salud es muy bajo y no se encuentran botiquines, el bot tiende a comportarse como si estuviera asustado, evitando cualquier riesgo.
BP6,4 Habilidad para mantener una representación precisa del cuerpo del jugador: El bot controla su posición, gesto y orientación de forma efectiva. Por ejemplo, es capaz de coordinar sus sistemas sensoriomotores para correr en una dirección a la vez que dispara en otra dirección, donde se encuentra un enemigo, gracias a un giro de cintura.
BP6,5 Habilidad para aprender conceptos abstractos relacionados con el juego: Las decisiones inteligentes que toma el bot indican que ha aprendido conocimiento específico acerca del juego. Por ejemplo, el bot tiende a atacar enemigos solitarios, mientras que huye de grupos de enemigos.

Nivel 7. Autoconsciente (Self-Conscious) - L7 = 0.00
Seleccionar/Deseleccionar Todos
BP7,1 Habilidad para mantener un modelo del yo y una representación de segundo orden de la relación entre el yo y la acción percibida en el juego: El comportamiento del bot indica que está presente un sentido del yo. Las decisiones no se toman simplemente en función del estado del jugador (salud, munición, etc.), sino que se basan en un modelo elaborado del yo que constituye la base de la capacidad de TdM.
BP7,2 Habilidad para mantener una representación análoga de segundo orden de la relación entre el yo y las acciones del bot: El bot es capaz de reconocerse a sí mismo y las consecuencias de sus propias acciones.
BP7,3 Habilidad para mantener una representación de segundo orden de la relación entre los sentimientos y el yo: las acciones también se evalúan en función de sus consecuencias para el propio yo.
BP7,4 Habilidad para auto-reconocerse como jugador y parte del juego: el bot pasa la prueba del espejo.
BP7,5 Habilidad para hacer planes incluyendo el modelo del yo como un actor en esos planes: El comportamiento se modula gracias a la habilidad del bot para prever (recordar) el resultado emocional de acciones conocidas.
BP7,6 Habilidad para imaginar el resultado para el yo de las acciones planificadas: El comportamiento del bot está modulado por su habilidad para prever (imaginar) el resultado emocional de una acción planificada. Por lo tanto aparecen nuevos comportamientos como resultado de este mecanismo de planificación avanzado. Por ejemplo, el bot desarrolla nuevas estrategias de ataque que no han sido aprendidas usando aprendizaje por refuerzo, sino que han sido imaginadas.
BP7,7 Habilidad para usar los objetos del juego como herramientas (dado que los juegos FPS proporcionan soporte nativo para el uso de armas y vehículos, su uso no puede considerarse como una capacidad cognitiva del bot): El bot es capaz de usar algunos objetos como medio para alcanzar sus objetivos. Por ejemplo, usando un objeto que se pueda mover, como una caja o barril, como un escudo improvisado.

Nivel 8. Empático (Empathic) - L8 = 0.00
Seleccionar/Deseleccionar Todos
BP8,1 Habilidad para modelar otros jugadores como yos intencionales con subjetividad: Dado que el bot puede modelar a otros jugadores como sujetos intencionales, puede predecir sus movimientos. El bot se pone (mentalmente) en el lugar de otros jugadores para poder prever las acciones de sus oponentes. El comportamiento del bot se modula no sólo en función de la información percibida, sino que también se tienen en cuenta los movimientos que previsiblemente realizará el oponente. Por ejemplo, el bot predice el posible camino de huida de un enemigo y hace el movimiento necesario para bloquearlo (incluso antes de que el oponente inicie su movimiento).
BP8,2 Habilidad para aprender de otros jugadores por imitación: Como el bot mantiene tanto un modelo del yo como un modelo de los otros, también puede establecer analogías y aprender estrategias observando otros bots. Por ejemplo, el bot puede adquirir nuevas estrategias de ataque llevadas a cabo por jugadores humanos que participan en el mismo juego.
BP8,3 Habilidad para colaborar con otros jugadores para obtener mejores puntuaciones: Los bots muestran comportamientos sociales como la formación de grupos que colaboran en los combates.
BP8,4 Habilidad para hacer planes incluyendo los modelos de los otros jugadores como actores en los planes (intersubjetividad): Los bots muestran comportamientos sociales como la formación de grupos que colaboran en los combates.
BP8,5 Habilidad para construir nuevas herramientas que se puedan usar para conseguir metas del juego: El bot es capaz de combinar varios objetos del escenario de juego con el objetivo de crear un objeto compuesto que sirva como defensa o ataque. Por ejemplo, el bot construye una barricada improvisada hecha de barriles colocados a lo largo de una línea.

Nivel 9. Social (Social) - L9 = 0.00
Seleccionar/Deseleccionar Todos
BP9,1 Habilidad para desarrollar estrategias Maquiavélicas como parte del juego: El bot es capaz de razonar sobre las capacidades de TdM de los oponentes. Por lo tanto, muestra comportamientos de inteligencia social, como preparar una emboscada.
BP9,2 Aprendizaje de nuevas estrategias Maquiavélicas: el bot aprende nuevas estrategias Maquiavélicas que no han sido pre-programadas, como nuevas formas de engañar a los enemigos.
BP9,3 Habilidad para comunicar el contenido mental: El bot usa el sistema de chat del juego para comunicar de forma coherente su estado mental explícito.
BP9,4 Habilidad para desarrollar una cultura: El perfil de comportamiento asociado con la cultura requeriría un entorno más complejo. Sin embargo, se podrían observar indicios de rasgos culturales en grupos de bots organizados.

Nivel 10. Androide (Human-Like) - L10 = 0.00
Seleccionar/Deseleccionar Todos
BP10,1 Habilidad para comunicar de forma precisa el contenido mental: el bot genera mensajes lingüísticos complejos usando una gramática correcta y un significado preciso.
BP10,2 Habilidad para superar una versión adaptada para juegos FPS del Test de Turing: El bot es capaz de superar un Test de Turing adaptado, como el propuesto en la competición BotPrize.
BP10,3 Habilidad para modificar el entorno y adaptarlo a las necesidades propias: Como en BP9,4 los comportamientos complejos asociados con estas habilidades requieren entornos más elaborados.
BP10,4 Habilidad para desarrollar una civilización y una tecnología: Como en BP9,4 los comportamientos complejos asociados con estas habilidades requieren entornos más elaborados.

Nivel 11. Super-Consciente (Super-Conscious) - L11 = 0.00
BP11,1 Habilidad para gestionar varios flujos de conciencia: Por ejemplo, el bot es capaz de mantener una conversación explícita compleja por el chat con un humano a la vez que combate a sus enemigos de forma efectiva.
